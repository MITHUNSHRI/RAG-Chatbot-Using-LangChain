# RAG Chatbot using LangChain and Qwen

This project demonstrates a basic Retrieval-Augmented Generation (RAG) chatbot built using:

- LangChain
- FAISS
- Hugging Face Embeddings
- Qwen 2.5 1.5B Instruct
- Google Colab

## Features

- Reads custom text documents
- Creates vector embeddings
- Stores embeddings in FAISS
- Retrieves relevant context
- Answers questions using a Large Language Model

## Technologies

- Python
- LangChain
- Transformers
- FAISS
- Sentence Transformers
- Google Colab

## Example Questions

- Who is the Principal?
- What are the library timings?
- Who is Mithun?
